For creating vector images using CorelDraw

CorelDRAW is a vector graphics editor used for creating and editing images. It is developed and marketed by Corel Corporation. CorelDRAW is used for a wide variety of graphic design purposes, including logo design, illustration, page layout, and web design.

CorelDRAW was first released in 1989 and has since gone through many versions. The latest version, CorelDRAW 2021, was released in March 2021. CorelDRAW is available for Windows and macOS operating systems.
